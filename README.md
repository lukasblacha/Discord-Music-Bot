# Discord-Music-Bot

Simple Music-Bot for Discord written in Python with <3

## Features
- Using modern /-Commands
- Can play/stream songs from YouTube
- Multi-Server ready

## Commandlist
> - /join
> - /play
> - /nowplaying
> - /resume
> - /stop
> - /stream
> - /volume
